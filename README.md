# ひとくせ博覧会

## サイト概要
日本全国にある風変わりでマニアックな、一癖ある博物館を共有できるアプリです。
同行するお仲間探しにも使えます。

### サイトテーマ
「あなたの知らない未知の世界へ」

### テーマを選んだ理由

### ターゲットユーザ
- 普通の博物館では物足りない方々
- 非日常を味わいたい方々

### 主な利用シーン
- マニアックな博物館へ行きたいと思ったときに
- 旅先で風変わりな経験をしたいと思ったときに
- 同行するメンバーの募集をしたいときに
